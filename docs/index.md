## 2. 生物信息学在做什么
我把它理解成：  
**拿到生物数据（序列/表达/图像等） → 用计算方法处理 → 得到能解释或能预测的结果。**

slides 里给了一个四步的框架：
1) **Question**：先想清楚问题是什么  
2) **Information**：有什么数据/信息  
3) **Analysis**：先把数据处理、统计、画图  
4) **Modeling**：需要的话再建模型（或者用现成模型）

---

## 3. Algorithm vs Model

### 3.1 一句话
- **算法（algorithm）**：一套“怎么算”的步骤（输入→步骤→输出）
- **模型（model）**：一套“怎么理解数据/怎么定义好坏”的假设（通常有参数）

### 3.2 我能想到的例子
- **模型**：线性回归、逻辑回归、朴素贝叶斯、HMM、神经网络  
  （共同点：都有参数，通常要从数据里估计/训练）
- **算法**：排序、动态规划  
  （共同点：更像固定步骤，关注效率和实现）

### 3.3 二者关系
- 模型：定“标准”（什么算好 / 最可能）  
- 算法：去“算出来”（怎么把答案找出来，最好还要快）
